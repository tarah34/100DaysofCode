Splitting a string into an array.

Another potentially useful method to use on a string.

Note the split method used on the "sonsString" array variable. This allows us to identify and output the "middle son". The "copyWithin" method ouputs the contents of the array.

  var sonsString = "Jesse, Josh, Aaron";
  var sonsAsArray = sonsString.split(",");
  document.write(sonsAsArray.copyWithin());
  document.write(" Middle son: " + sonsAsArray[1]);
