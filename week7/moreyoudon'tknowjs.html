More You Don't Know JS book practice/coding

Values and Types. There are 6 types:
  1. string
  2. number
  3. boolean
  4. null & undefinded
  5. object
  6. symbol (new to ES6)

Comparing Values

  • explicit

    • var a = "42";

    var b = Number(a);

    a; // "42"

    b; // 42 - the number!

  • implicit

    • var a = "42"

    var b = a * 1;

    a; // "42" //\\ b; // 42 - the number!

In JavaScript, objects are king. If you understand objects, you understand JavaScript.
